║                          📝 How to Add Keys                                  ║
╚══════════════════════════════════════════════════════════════════════════════╝

1. Open file: backend/.env

2. Replace placeholders:
   OPENAI_API_KEY=sk-your-actual-key-here
   NEWS_API_KEY=your-actual-key-here

3. Save file

4. Restart backend server
