## 🚀 How to Use

### Quick Start (5 Minutes)

1. **Run Setup Script**
   ```bash
   # Windows
   start.bat
   
   # Linux/Mac
   chmod +x start.sh
   ./start.sh
   ```

2. **Access Application**
   - Frontend: http://localhost:3000
   - Backend: http://localhost:8000
   - API Docs: http://localhost:8000/docs

3. **Try Demo Queries**
   ```
   "Which stocks should I buy?"
   "Predict AAPL price"
   "Create portfolio ₹50000"
   ```
